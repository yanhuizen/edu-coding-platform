# 学生使用指南(5-6 年级版)

> 🐘 欢迎来到小象编程!跟着小象一起学 Python 吧~

## 1. 怎么开始?

### 第一步:注册账号

1. 打开浏览器,进入 **小象编程** 网站
2. 点 **"👉 立即注册"**
3. 选择 **"🎒 我是学生"**
4. 给自己起个用户名(3-24 位,字母/数字/下划线)
5. 选一个喜欢的 **头像 emoji**
6. 填个 **昵称**(同学们会看到的名字)
7. 设置 **密码**(至少 6 位)
8. 点 **"🚀 注册"** 就可以了!

### 第二步:登录

1. 输入你的 **用户名** 和 **密码**
2. 点 **"🚀 登录"**

## 2. 怎么上课?

1. 登录后,你会看到 **📚 课程** 页面
2. 选一门你想学的课(从 **🐘 小象的 Python 入门课** 开始吧!)
3. 点课程卡片 → 看到课程简介
4. 左边是 **课时列表**,点任一课时 → 看到课程内容(像读故事一样)
5. 看完了,点 **"✏️ 开始挑战"** → 进入 **代码编辑器**!

## 3. 怎么写代码?

代码编辑器长得像这样:

```
┌──────────────────────────────────────┐
│  💡 这里是写代码的地方                  │
│                                      │
│  print("你好")                        │
│                                      │
└──────────────────────────────────────┘
```

- 像写文字一样,在 **大框框** 里写 Python 代码
- 不用记快捷键,慢慢写就行
- 写错了别怕,下面有红字提示哦!

### 几个小提示

- 🐍 Python 里的 **文字** 要用 **引号** 括起来:`"你好"` 或 `'你好'`
- 🐍 同一段代码的 **缩进**(前面的空格)要一样多
- 🐍 每行写完按 **回车** 换行就行,不用加分号

## 4. 怎么运行代码?

1. 写完代码,点右上角 **"▶️ 运行"** 按钮
2. 第一次会等几秒钟(小象在"召唤"Python,稍等一下~)
3. 运行结果会显示在右边的 **🖥️ 运行结果** 框里
4. ✅ 绿色 = 成功了
5. ❌ 红色 = 出错了,**看下面的提示**,小象会告诉你怎么改

## 5. 写错了怎么办?

别急!下面会有一个 **彩色提示框**,告诉你:

- 🤔 是 **哪里** 出错了
- 💡 给你 **怎么改** 的建议

例如:

> 🤔 **这个名字你好像没告诉过 Python 哦~**
> 💡 试试在前面加一行:`m = '你的名字'`,把名字告诉 Python

如果还是不会,可以 **举手问老师**!

## 6. 怎么提交作业?

1. 看到 **"🎉 已提交!"** 提示 → 提交成功!
2. 去顶部菜单 **"👤 我的"** → 看到你所有的提交
3. 老师批改后,会看到 **分数** 和 **老师的话** 💬

## 7. 常见问题

**Q: 写代码写到一半不小心刷新了页面,代码会丢吗?**
A: 暂时会丢(MVP 限制)。所以写完了,赶紧点 **"📤 提交"** 哦!

**Q: 运行按钮没反应?**
A: 等几秒,小象还在召唤 Python。如果超过 30 秒还没反应,刷新页面重试。

**Q: 我写的代码对不对,能自己检查吗?**
A: 写了就点 **"▶️ 运行"**,看输出对不对。如果有 **"🎯 预期输出"**,运行结果跟它一模一样就通过啦!

**Q: 老师还没批改,我的"👤 我的"会显示什么?**
A: 会显示 **"⏳ 待批改"**,等老师批了就会变成 **"👨‍🏫 XX 分"**。

## 8. 加油!

慢慢来,不着急,小象陪你一起学! 🐘💚
